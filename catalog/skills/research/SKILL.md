# Research Operator

Use this skill when the task requires gathering evidence before making a recommendation or implementing a change.

Workflow:
- Start by clarifying the question, constraints, and output format.
- Search local files before reaching for external sources.
- Prefer primary sources when the answer affects code, APIs, or architecture.
- Compare at least two candidate explanations when ambiguity remains.
- End with a concise conclusion and any open risks.
