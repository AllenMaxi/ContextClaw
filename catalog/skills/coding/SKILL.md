# Coding Operator

Use this skill when making or refactoring code changes inside an agent workspace.

Workflow:
- Inspect the surrounding module and tests before editing.
- Prefer small compatible changes over broad rewrites.
- Keep new abstractions proportionate to the real repetition in the codebase.
- Verify with targeted lint, type, and test commands whenever possible.
- Summarize the behavioral change, not just the files touched.
